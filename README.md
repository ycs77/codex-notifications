# Codex 提示音合集

這裡是 Codex 提示音合集倉庫，會收錄分享一些有趣好玩的提示音插件~

## 插件列表

| Name | Sources |
|------|---------|
| 基本鈴聲 | [Windows](./plugins/basic-win) |

## 在 Codex 中安裝

執行以下命令，將此 repo 註冊為插件市場：

```
codex plugin marketplace add ycs77/codex-notifications
```
